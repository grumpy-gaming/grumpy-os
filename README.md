# 🦉 Grumpy-OS  
*A gaming-optimized Linux-based OS built for performance and customization.*

## 🔥 Overview  
Grumpy-OS is a custom fork of KNULLI designed for handheld gaming devices. It blends performance tweaks and an intuitive frontend for seamless gameplay.

## 🚀 Features  
- **Optimized Kernel & Firmware** – Tweaked for stability and speed.  
- **EmulationStation Frontend** – Integrated for easy game browsing.  
- **Lightweight & Minimalist** – Stripped-down for fast execution.
